# BYW Backend API

Production-ready backend for Bid Your Work platform.

## Features
- JWT Authentication with role-based access
- Real-time chat & notifications via Socket.io
- Wallet & payment system with transaction history
- Job bidding with time limits
- Rating system
- MongoDB persistence

## Setup
1. Clone repository
2. `npm install`
3. Copy `.env.example` to `.env` and configure
4. `npm run dev` for development

See API endpoints list for complete details.
